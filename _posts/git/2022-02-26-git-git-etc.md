# 💡 좋은 커밋 메세지의 7가지 규칙
    1. 제목과 본문을 빈 줄로 분리한다.
    2. 제목은 50자 이내로 작성한다.
    3. 제목을 영어로 작성할 경우, 첫 글자는 대문자로 작성한다.
    4. 제목에는 마침표를 작성하지 않는다.
    5. 제목을 영어로 작성할 경우, 동사원형(현재형)으로 시작한다.
    6. 본문을 72자 단위로 줄바꿈한다.
    7. How보다 What과 Why를 설명한다.

## 💡 etc
