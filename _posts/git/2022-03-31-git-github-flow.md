# 📚 Introduction

업무시간 이외에 혼자서 간단한 토이프로젝트를 시작해볼까 고민하던 찰나, <br>
지인들과 토이프로젝트를 같이 하게 되었다. 👍🙌 <br>
전에 git의 기본개념과 git `GUI`에 대한 대략적인 사용방법에 대해서만 한 번 훑어보기만 했었는데, <br> 
이번 프로젝트를 통해서 git을 `CLI` 환경에서 사용하는데에 조금이나마 익숙해질 수 있는 기회가 될 것 같다. 🤩
